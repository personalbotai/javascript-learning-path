Operator dan Ekspresi

Kategori Operator

  • Arithmetic + - * / % **
  • Assignment = += -= *= /= %= **=
  • Comparison == === != !== > < >= <=
  • Logical && || ! ??
  • Bitwise & | ^ ~ << >> >>>
  • String + (concat)
  • Conditional ? : (ternary)
  • Unary + - ~ ! typeof void delete
  • Comma ,

Arithmetic Operators

10 + 5;     // 15
10 - 5;     // 5
10 * 5;     // 50
10 / 5;     // 2
10 % 3;     // 1 (remainder)
2 ** 3;     // 8 (exponentiation, ES2016)

Division by zero:

5 / 0;   // Infinity
-5 / 0;  // -Infinity
0 / 0;   // NaN

Assignment & Compound Assignment

let x = 10;
x += 5;   // x = 15
x -= 3;   // x = 12
x *= 2;   // x = 24
x /= 4;   // x = 6
x %= 5;   // x = 1
x **= 3;  // x = 1

Comparison Operators

Operator Description Example Result
== Loose equality (type coercion) 5 == "5" true
=== Strict equality (no coercion) 5 === "5" false
!= Loose inequality 5 != "6" true
!== Strict inequality 5 !== "5" true
> Greater than 10 > 5 true
< Less than 3 < 7 true
>= Greater or equal 5 >= 5 true
<= Less or equal 5 <= 5 true

⚠️ Gunakan === dan !== selalu, hindari == dan !=.

Logical Operators

// && (AND) - kedua sisi harus truthy
true && true;     // true
true && false;    // false
"hello" && 42;    // 42 (return last truthy)

// || (OR) - return first truthy, atau last falsy
true || false;    // true
false || "hi";    // "hi"
0 || "" || null;  // null (semua falsy)

// ! (NOT) - boolean negation
!true;            // false
!false;           // true
!"hello";         // false
!0;               // true
!!"test";         // true (force boolean)

Nullish Coalescing (??)

Return right operand jika left adalah null atau undefined (tidak termasuk falsy lain seperti 0 atau "").

0 ?? 100;       // 0 (0 bukan nullish)
"" ?? "default"; // ""
null ?? "default"; // "default"
undefined ?? 42;   // 42
let name = null;
console.log(name ?? "Anonymous"); // "Anonymous"

Unary Operators

// typeof - cek tipe data (string result)
typeof 42;          // "number"
typeof "hi";        // "string"
typeof true;        // "boolean"
typeof undefined;   // "undefined"
typeof null;        // "object" (bug!)
typeof [1,2];       // "object"
typeof function(){} // "function"

// void - selalu return undefined
void 0;             // undefined
void (console.log("hi")); // undefined, tapi log executed

// delete - hapus property dari object
let obj = {a:1, b:2};
delete obj.a;       // true
console.log(obj);   // {b:2}

// + (unary plus) - convert ke number
+"42";              // 42
+"3.14";            // 3.14
+true;              // 1
+false;             // 0
+"hello";           // NaN

// - (unary minus) - negate number
-5;                 // -5
-"10";              // -10
-(-5);              // 5

Operator Precedence

Operator memiliki prioritas. Gunakan parentheses untuk kejelasan.

// Precedence tinggi: **
2 ** 3 ** 2;  // 512 (2^(3^2) = 2^9)

// * / % sebelum + -
10 + 20 * 5;  // 110 (20*5 dulu)

// && sebelum ||
true || false && false; // true (&& dulu)

// Gunakan parentheses untuk explicit grouping
(10 + 20) * 5; // 150

// Conditional (ternary) operator
let status = score >= 60 ? "pass" : "fail";

Short-Circuit Evaluation

Logical operators && dan || tidak mengevaluasi operand kedua jika hasil sudah bisa ditentukan dari operand pertama.

AND Short-Circuit (&&)

function test() {
    console.log("test executed");
    return true;
}

false && test();  // test() TIDAK dieksekusi (false already)
true && test();   // test() dieksekusi, return true

OR Short-Circuit (||)

function compute() {
    console.log("compute executed");
    return 42;
}

true || compute();   // compute() TIDAK dieksekusi
false || compute();  // compute() dieksekusi, return 42

Praktik Umum: Default Values

// || untuk fallback (tapi hati-hati dengan falsy values seperti 0, "")
const name = inputName || "Anonymous";
const count = items || [];

// ?? untuk nullish fallback (lebih aman)
const name = inputName ?? "Anonymous";
const count = items ?? [];
