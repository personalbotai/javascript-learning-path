Variabel dan Tipe Data

Deklarasi Variabel: var, let, const

JavaScript memiliki 3 cara mendeklarasi variabel. Gunakan let dan const untuk hampir semua kasus.

Keyword Scope Reassign Hoisting Rekomendasi
var Function scope ✅ ✅ (undefined) Hindari
let Block scope ✅ ❌ (Temporal Dead Zone) Nilai yang berubah
const Block scope ❌ ❌ (Temporal Dead Zone) Default pilihan

Contoh:

let age = 25;      // bisa diubah
age = 26;          // ✅ OK

const name = "Budi"; // tidak bisa reassign
name = "Ana";        // ❌ TypeError

// Tapi const untuk object/array: isi bisa diubah
const arr = [1, 2, 3];
arr.push(4);  // ✅ OK (mutasi)
arr = [5, 6]; // ❌ TypeError (reassign)

Tipe Data Primitif

JavaScript memiliki 7 tipe primitif:

Number

Integer dan floating point. Batas: ±(2^53-1). Infinity, NaN juga number.

42; 3.14; -7; NaN; Infinity

BigInt

Integer dengan ukuran arbitrer (ditambahkan ES2020). Akhiri dengan n.

9007199254740991n; BigInt("12345678901234567890")

String

Teks dengan single/double/backtick quotes. Template literals dengan backtick.

"Hello"; 'World'; `Value: ${x}`

Boolean

Nilai logika: true atau false.

true; false

undefined

Nilai yang belum didefinisikan. Tipe dari undefined adalah "undefined".

let x; console.log(x); // undefined

null

Nilai "tidak ada" yang sengaja dikelompokkan. Tipe null adalah "object" (bug sejarah).

let y = null;

Symbol

Identifier unik. Berguna sebagai property key yang unik.

const id = Symbol("id");
let obj = { [id]: 123 };

Tipe Referensi (Object)

Semua yang bukan primitif adalah object, termasuk:

  • Object - {} atau new Object()
  • Array - [] atau new Array() (ordered list)
  • Function - function() {} atau arrow () => {}
  • Date - new Date()
  • RegExp - /pattern/ atau new RegExp()
  • Map, Set, WeakMap, WeakSet - collections

Catatan: Tipe referensi disimpan sebagai referensi di memory, bukan nilai copy.

let a = [1, 2, 3];
let b = a;     // b merefer ke array yang sama
b.push(4);
console.log(a); // [1,2,3,4] - ikut berubah!

Type Checking

Gunakan typeof untuk tipe primitif dan instanceof untuk object.

typeof 42;           // "number"
typeof "hello";     // "string"
typeof true;        // "boolean"
typeof undefined;   // "undefined"
typeof null;        // "object"  ⚠️ bug sejarah
typeof NaN;         // "number"
typeof [1,2];       // "object"
typeof {a:1};       // "object"
typeof function(){} // "function"

// instanceof untuk specific object type
[1,2] instanceof Array;      // true
new Date() instanceof Date;  // true
/abc/ instanceof RegExp;     // true

Untuk null check, gunakan strict equality: value === null.

Type Coercion & Conversion

JavaScript otomatis mengkonversi tipe dalam operasi tertentu. Ini bisa membingungkan.

Implicit Coercion (otomatis)

"5" + 3;   // "53" (string concat)
"5" - 3;   // 2 (string to number)
"5" * "3"; // 15
"5" == 5;  // true (type coercion)
"5" === 5; // false (strict equality)

Explicit Conversion (disarankan)

Number("5");      // 5
parseInt("10");    // 10
parseFloat("3.14");// 3.14
String(42);        // "42"
Boolean(0);        // false
Boolean("");       // false
Boolean("hello");  // true

Selalu gunakan === (strict equality) kecuali Anda tahu pasti perlu coercion.

Truthy & Falsy

Nilai berikut selalu falsy dalam konteks boolean:

  • false
  • 0
  • -0
  • 0n (BigInt zero)
  • "" (empty string)
  • null
  • undefined
  • NaN

Semua nilai lain adalah truthy:

if ("0") console.log("truthy"); // "0" adalah truthy!
if ([]) console.log("truthy");   // array kosong truthy
if ({}) console.log("truthy");   // object kosong truthy
if (42) console.log("truthy");   // angka selain 0 truthy
