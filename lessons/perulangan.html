Perulangan: for, while, do-while

Mengapa Perulangan?

Perulangan memungkinkan kita mengeksekusi blok kode berulang kali tanpa menulis kode yang sama berulang. Ini adalah konsep fundamental untuk:

  • Iterasi melalui array atau object
  • Melakukan task berulang (misal: validasi input)
  • Menggenerate output
  • Algoritma (sorting, searching, dll)

1. For Loop (Count-controlled)

Digunakan ketika tahu berapa kali iterasi diperlukan.

// Syntax dasar
for (initialization; condition; increment) {
    // body
}

// Contoh: cetak angka 1-5
for (let i = 1; i <= 5; i++) {
    console.log(i);
}
// Output: 1 2 3 4 5

Variasi For Loop

// Multiple variables
for (let i = 0, j = 10; i < j; i++, j--) {
    console.log(i, j);
}
// 0 10, 1 9, 2 8, ..., 4 6

// Omit parts (hati-hati!)
let i = 0;
for (; i < 5; ) {
    console.log(i++);
}

// Infinite loop (jangan lupa break!)
for (;;) {
    console.log("infinite");
    break; // necessary!
}

// For...of (iterate values) - ES6
const arr = [10, 20, 30];
for (const value of arr) {
    console.log(value); // 10, 20, 30
}

// For...in (iterate keys) - untuk object
const obj = {a:1, b:2, c:3};
for (const key in obj) {
    console.log(key, obj[key]); // "a" 1, "b" 2, "c" 3
}

2. While Loop (Condition-controlled)

Loop selama kondisi true. Cocok ketika jumlah iterasi tidak diketahui sebelumnya.

while (condition) {
    // body
}

// Contoh: random number sampai dapat 5
let roll;
do {
    roll = Math.floor(Math.random() * 6) + 1;
    console.log("Rolled:", roll);
} while (roll !== 5);

Do-While Loop

Body selalu dieksekusi minimal sekali, kondisi dicek setelahnya.

do {
    // body (always runs at least once)
} while (condition);

// Contoh:
let input;
do {
    input = prompt("Enter a number (0 to quit):");
    console.log("You entered:", input);
} while (input !== "0");

Infinite Loop & Performance

Loop yang tidak pernah berakhir akan membeku browser/Node.js. Pastikan kondisi akan menjadi false suatu saat.

// DANGER: Infinite loop
while (true) {
    console.log("This never stops!");
}

// BETTER: Ensure condition changes
let counter = 0;
while (counter < 100) {
    console.log(counter++);
}

// Avoid heavy operations in loop condition
for (let i = 0; i < array.length; i++) { /* OK */ }
// vs
for (let i = 0; i < array.length; i++) { array.length = expensiveOperation(); } // BAD

Control Flow: break & continue

break

Langsung keluar dari loop.

for (let i = 0; i < 10; i++) {
    if (i === 5) break;
    console.log(i); // 0 1 2 3 4
}

continue

Lewatkan iterasi saat ini, lanjut ke berikutnya.

for (let i = 0; i < 5; i++) {
    if (i === 2) continue;
    console.log(i); // 0 1 3 4
}

Iterasi Through Collections

Array: for...of (recommended)

const numbers = [1, 2, 3, 4, 5];
for (const num of numbers) {
    console.log(num * 2);
}

// With index
for (const [index, value] of numbers.entries()) {
    console.log(`Index ${index}: ${value}`);
}

Array: forEach (functional)

numbers.forEach((num, index) => {
    console.log(`numbers[${index}] = ${num}`);
});

Object: for...in

const person = {name: "Budi", age: 25, city: "Jakarta"};
for (const key in person) {
    console.log(`${key}: ${person[key]}`);
}

// Object.keys() + forEach
Object.keys(person).forEach(key => {
    console.log(key, person[key]);
});

String: for...of

const str = "Hello";
for (const char of str) {
    console.log(char); // H e l l o
}

Common Loop Patterns

// 1. Loop backwards (reverse)
for (let i = arr.length - 1; i >= 0; i--) {
    console.log(arr[i]);
}

// 2. Loop with step
for (let i = 0; i < 100; i += 10) {
    console.log(i); // 0, 10, 20, ..., 90
}

// 3. Nested loops (2D array)
const matrix = [[1,2], [3,4], [5,6]];
for (let row of matrix) {
    for (let cell of row) {
        console.log(cell);
    }
}

// 4. While with array (pop all)
while (arr.length) {
    console.log(arr.pop());
}

// 5. Do-while for menu
let choice;
do {
    choice = prompt("1. Play 2. Settings 3. Exit");
    // handle choice
} while (choice !== "3");
